Behind The Curtain

HTML is the tool used to create this website and all others on the internet. Even more complicated tools like PHP and Microsoft's Blazor become plain old HTML by the time they reach your browser.

HTML is rarely used alone, though -- it is almost always paired with CSS to do the styling and JavaScript to make the site reactive (e.g. buttons most animations).

There are many other additional tools you can use to make writing websites easier and so, so much harder at the same time, such as:

  • React
  • Bootstrap
  • JSX
  • JQuery
There are tools that extend the functionality of websites using dedicated servers, like:
  • PHP
  • Blazor
  • Node.js
For this website, however, we chose to use only vanilla HTML, CSS, and JavaScript in a misguided and quickly abandonded attempt to keep things simple. Read on to learn how we did it!
